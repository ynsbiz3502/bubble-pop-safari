# 🫧🐾 Bubble Pop Safari

A cheerful, self-contained bubble-popping game for **preschoolers (ages 3–5)**, inspired by the friendly, encouraging style of *Reader Rabbit Kindergarten/Toddler*.

**▶️ Play it here:** https://ynsbiz3502.github.io/bubble-pop-safari/

## What it does
- 🫧 **Pop floating bubbles** with a tap (works great on tablets & touchscreens).
- 🐰 Each bubble holds an **animal, fruit, or fun object** that bursts out with a bouncy animation.
- 🔊 **Sound & voice** — a friendly voice says the name of each thing you pop ("Cat!", "Apple!"), plus playful pop sounds and gentle background music. *(All sound is generated in-browser — no files, works offline.)*
- ✨ **Visual effects** — colorful particle bursts, confetti, floating words, and ripples.
- ⭐ **Gamified** — collect stars, fill the prize jar, level up, and earn stickers for a sticker book.
- 🌈 **Special bubbles** — gold and rainbow bubbles give bonus stars and extra sparkle.
- 😊 **Stress-free** — no fail states, no timers, endless friendly play. Missed bubbles just float away.

## Learning goals (Reader Rabbit style)
- Hearing and connecting **names** with pictures (early vocabulary).
- **Counting** stars and tracking progress.
- Cause-and-effect, fine-motor tapping, color recognition.

## Tech
- A single `index.html` — **no build step, no dependencies, no external assets.**
- Web Audio API for sound, SpeechSynthesis for voice, Web Animations API for effects.
- Hosted free on **GitHub Pages**.

Made with ❤️ for little hands.
